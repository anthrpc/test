HTML Basics

HTML (Hypertext Markup Language) is a language used for structuring and formatting the content of a webpage. It allows you to create and organize text, images, and other elements on a webpage using a set of tags and attributes.

What are tags and attributes?

HTML tags are used to mark the beginning and end of an element on a webpage. They are enclosed in angle brackets, like this: <tagname>. Attributes provide additional information about an element and are used to customize the way it is displayed. They are added to the opening tag, like this: <tagname attribute="value">.

HTML elements are the building blocks of an HTML document. They are used to define the content and structure of a webpage.

Example: Creating a paragraph

To create a paragraph in HTML, you can use the <p> tag. For example:

<p>This is a paragraph.</p>

This will produce the following output:

This is a paragraph.

    1. Unordered lists: To create an unordered list, you can use the ul element. Each list item should be wrapped in li tags.

    Code Snippet: Creating a Simple List

    To create a simple list in HTML, you can use the `ul` element to create an unordered list and the `li` element to add list items. Here is an example:

    <li>Item 3</li>
  </ul>

    • Item 1
    • Item 2
    • Item 3

    HTML Headings

    HTML has six levels of headings, which are used to indicate the importance of a section of content. The highest level heading is <h1> and the lowest level heading is <h6>.

    Here is an example of how to use headings in HTML:

          <h1>Heading Level 1</h1>
      <h2>Heading Level 2</h2>
      <h3>Heading Level 3</h3>
      <h4>Heading Level 4</h4>
      <h5>Heading Level 5</h5>
      <h6>Heading Level 6</h6>

    HTML Tables

    HTML tables are used to display data in a tabular format, with rows and columns. They are defined using the table element, and each row is defined using the tr element. Cells within a row can be defined using the td element (standard cell) or the th element (table header cell).
